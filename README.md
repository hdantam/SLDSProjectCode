# SLDSProjectCode
Code for our statisical analysis project

##The folders are divided into 3 sections:
### 1. Data_reading_and_creation
This was code needed to read in and get the coordinate data for the various datasets (MRT, Bus, HDB)

### 2. Datasets
A comprehensive folder of all the datasets involved in our project, with a key folder showing which we used for our final modelling and testing


### 3. Data_analysis_modelling
This is the code involved in our exploratory analysis and modelling
